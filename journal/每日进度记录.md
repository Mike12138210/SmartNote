```
遇到的问题与解决
- 发送申请成功后如何返回不同提示 → 通过 `Result.success(String message, T data)` 重载实现
- 搜索用户时 `or` 与 `and` 优先级问题 → 使用 `wrapper.and(w -> w.eq(...).or().eq(...).or().like(...))` 解决
- 用户资料修改时 `update_time` 不更新
  - 原因：MyBatis-Plus 默认只更新非null字段，`updateTime` 有旧值导致被忽略
  - 解决方案：在实体类中使用 `@TableField(update = "now()")`，让数据库自动更新时间，替代自动填充处理器
```
